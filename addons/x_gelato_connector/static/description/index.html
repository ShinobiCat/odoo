Odoo Gelato Connector

De Odoo Gelato Connector is een module waarmee je producten kunt importeren en bestellingen kunt plaatsen vanuit je Odoo e-commerce winkel naar Gelato, een print-on-demand service voor het afdrukken en leveren van op maat gemaakte producten.

Kenmerken:

  • Importeer producten van Odoo naar Gelato
  • Plaats bestellingen voor op maat gemaakte producten bij Gelato
  • Integratie met Odoo e-commerce winkel

Gebruiksinstructies:

  1. Installeer en configureer de module in Odoo
  2. Verbind de Gelato API-sleutel en basis-URL met de module
  3. Importeer producten naar Gelato vanuit de Odoo e-commerce winkel
  4. Plaats bestellingen voor op maat gemaakte producten bij Gelato vanuit Odoo
